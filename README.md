# IsEven_python
to find if a number is even or not

## How the code is written

```python
def is_Even(number):
    if number == 1:
        return False
    elif number == 2:
        return True
    elif number == 3:
        return False
    #and so on....
```
## To run the code

```python
# Eg lets take for 10
print(is_Even(10))
```
