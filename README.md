# IsEven_python
A very inefficient code to check if a number is even or odd

## An efficient version of the code

```python
def is_Even(number):
    if number % 2 == 0:
        return True
    else:
        return False
```

## How the code is written

```python
def is_Even(number):
    if number == 1:
        return False
    elif number == 2:
        return True
    elif number == 3:
        return False
    #and so on....
```
## To run the code

```python
# Eg lets take for 10
print(is_Even(10))
```
